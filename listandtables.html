HTML : Nested Lists and Tables

Introduction

This covers structuring complex data using nested lists for
hierarchical information and tables for grid-based data.

- Nested Lists: Placing a list inside of another list item.
- Tables: Organizing data into a grid of rows and columns.

1. Nested Lists (<ul>, <ol>, <li>)

A list placed inside another list to create a hierarchy.

Example (Unordered in Unordered):
<ul>
  <li>France
    <ul>
      <li>Paris</li>
    </ul>
  </li>
</ul>

Example (Ordered in Unordered):
<ul>
  <li>Car
    <ol>
      <li>Mercedes</li>
      <li>BMW</li>
    </ol>
  </li>
</ul>

2. Tables (<table>, <tr>, <td>, <th>)

Used to display data in a grid.

- <table>: The main container.
- <tr>: Defines a table row.
- <td>: Defines a standard data cell.
- <th>: Defines a header cell (bold and centered).

Example:
<table border="2">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Rohit</td>
    <td>25</td>
  </tr>
</table>

3. Advanced Table Structure

These tags group table content for better structure.

- <thead>: Groups header content.
- <tbody>: Groups body content.
- <tfoot>: Groups footer content.

Example:
<table>
  <thead>
    <tr>
      <th>Item</th>
      <th>Price</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Laptop</td>
      <td>$1200</td>
    </tr>
  </tbody>
</table>

4. Spanning Columns & Rows

Attributes to make a cell extend over multiple cells.

- colspan: Merges a cell with cells to its right.
- rowspan: Merges a cell with cells below it.

Example:
<table>
  <tr>
    <th colspan="2">Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td rowspan="2">25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>Smith</td>
  </tr>
</table>
