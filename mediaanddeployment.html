HTML : Media & Special Characters

1. Embedding Media (<video> & <audio>)

HTML5 introduced native tags for media, removing the need for plugins
like Flash. The <video> and <audio> tags are used to embed
media directly into a webpage.

The controls attribute is essential. It's a boolean attribute that
adds a full set of player controls (play, pause, volume, etc.).

Video Example:
<video src="my-awesome-video.mp4" controls></video>

Audio Example:
<audio src="Mahol pura wavy.mp3" controls></audio>

Play Maahol Thoda Wavyyy 🌊 🌊

2. Common Media Attributes

You can control the behavior and appearance of the media player.

- width/height: Sets the dimensions of the video player.
  <video src="vid.mp4" width="640" height="360" controls></video>

- autoplay/muted: Starts the video automatically, but must be muted
  to work in modern browsers.
  <video src="vid.mp4" autoplay muted></video>

- loop: Restarts the media from the beginning when it finishes.

- poster: Displays an image thumbnail before the video is played.
  <video src="vid.mp4" poster="thumbnail.jpg" controls></video>

3. The <source> Element (Compatibility)

Not all browsers support the same video formats (.mp4, .webm, .ogg).
To ensure your video works everywhere, provide multiple formats using
the <source> tag. The browser will play the first one it supports.

This is the professional and most reliable way to embed video.

Example:
<video controls>
  <source src="my-video.webm" type="video/webm">
  <source src="my-video.mp4" type="video/mp4">
  Sorry, your browser doesn't support embedded videos.
</video>

4. Accessibility with <track>

To make videos accessible to a wider audience, use the <track> tag
to add captions (for hearing impaired) or subtitles (for translation).
These point to a text file in WebVTT (.vtt) format.

Example:
<video controls>
  <source src="video.mp4" type="video/mp4">
  <track src="captions_en.vtt" kind="captions" srclang="en" label="English">
</video>

HTML <Iframe> Syntax

The HTML <iframe> tag specifies an inline frame.
An inline frame is used to embed another document within the current HTML document.

Example:
<iframe src="demo_iframe.htm" style="height:200px;width:300px;" title="Iframe Example" allowfullscreen frameborder="0"></iframe>

5. HTML Entities (Special Symbols)

To display reserved characters or symbols, use HTML entities. They
can be written as a name or a number.

- &copy; or &#169; → © (Copyright)
- &reg;  or &#174; → ® (Registered Trademark)
- &lt;   or &#60;  → < (Less-than)
- &gt;   or &#62;  → > (Greater-than)

You must use entities for '<' and '>' to prevent the browser from
thinking they are part of an HTML tag.

Deployment

  1. Drag & Drop (Quick Way)
      - Go to Netlify Dashboard → Add new site → Deploy manually.
      - Drag and drop your folder (site root or build folder).
      - Done! You get a live URL.

   2. Deploy via Git (Recommended)
      - Push your code to GitHub.
      - Netlify Dashboard → Add new site → Import existing project.
      - Connect your repo.
      - Set:
        • Branch to deploy: main (or master)
      - Click Deploy site.
      - Done! URL will be live.
