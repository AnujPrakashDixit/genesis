HTML : Hyper Text Markup Language

Introduction

HTML stands for HyperText Markup Language. It is the standard language used 
to create and structure the content of a webpage.

- HyperText: The ability to link web pages together.
- Markup Language: Using "tags" to give meaning and structure to text.

First Website: https://info.cern.ch/hypertext/WWW/TheProject.html

Setting Up

1. Install Visual Studio Code → https://code.visualstudio.com/
2. Install Extensions:
   • Live Server
   • Prettier (for clean code formatting)

1. Headings (<h1> to <h6>)

Headings define titles and subtitles, creating a logical hierarchy.
Purpose: To give content structural importance, not just to make text large.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
...
<h6>Heading 6</h6>

2. Paragraphs & Breaks (<p>, <hr>, <br>)

- <p>: The paragraph tag groups sentences into a block of text.
  Browsers automatically add space before and after.

- <hr>: The horizontal rule tag creates a thematic break or a visible line.

- <br>: The line break tag forces text onto the next line.

Example:
<p>This is the first paragraph.</p>
<hr>
<p>221B Baker Street<br>London, England</p>

3. Lists (<ul>, <ol>, <li>)

Used to group related items together.

<ul> (Unordered List): Creates a bulleted list.
<ul>
  <li>Tea</li>
  <li>Sugar</li>
</ul>

<ol> (Ordered List): Creates a numbered list.
<ol>
  <li>Boil water</li>
  <li>Add tea</li>
</ol>

4. Anchor Links (<a>)

The anchor tag creates hyperlinks to other pages or files.

- href: The 'hypertext reference' or destination URL. (Required)
- target: Specifies where to open the link.
  - "_self": (Default) Opens in the same tab.
  - "_blank": Opens in a new tab.

Example:
<a href="https://www.google.com" target="_blank">Go to Google</a>

5. Images (<img>)

The image tag is a self-closing tag used to embed images.

- src: The 'source' path or URL to the image file. (Required)
- alt: 'Alternative text' for accessibility and if the image fails to load.
- width/height: Sets the dimensions of the image.

Example:
<img src="bhaiya.jpg" width="300" height="300" alt="Hero Ki Photo">
